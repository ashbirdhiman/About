Smart Dustbin using Aws and Machine Learning


Smart Dustbin using AWS is a waste segregation project which is
used to segegrate into biodegradable and non-biodegrable waste.
Raspberry Pi is used as small computer which is used to capture
images of waste and send to the AWS as an API and AWS analyze
the waste using CNN model and send the type of waste detected.
Raspberry Pi also send the realtime data to the AWS for managing
whether the dustbin is completely filled or not.


Skill Used:Raspberry-Pi, AWS,Python,Machine Learning

Exam Seating Arrangement using Data Mining


Seating Arrangement using data mining which is developed in JAVA
MySQL wamp server. This project is used to design and implement
exam seating arrangement for thapar polytechnic college.It is
dynamic seating arrangement as this uses a pdf file converted to
.csv file which contains rollno with courses as alloted by the PSBTE
board. There are pre-alloted room arrangement in the database with
varying capacity and different branches have different colour
which help the student to find their seat alloted easily.


Skill Used:Java,NetBeans,MySQL,Microsoft visual studio.




Smart Parking System


This project offers a desktop application based parking reservation
system where customers can view various parking spaces and select nearby or
specific area of their choice to view whether space is available or not.
If the booking space is available, then customers can book it for specific
time slot. The booked space will be marked and will not be available for anyone
else for the specified time. This system provides an additional feature of
cancelling the bookings. Customers can cancel their reserved space anytime.
Customers can also view previous parking booking details. Also image processing
is used to check the number plates and then validate that whether the data is
stored in a database or not? So this project will also serve as a theft control system.


Skill Used:Java,NetBeans,MySQL,Python,Pytesseract.
